=== Payment Gateways Country Limiter ===
Author: OnTheGoSystems
Tags: woocommerce, payment gateways, checkout
Requires at least: 3.8
Tested up to: 3.8.1
Requires WooCommerce at least: 2.0.20
Tested WooCommerce up to: 2.1.5

Allows showing checkout payment options according to the client's billing country.

See http://docs.woothemes.com/document/payment-gateways-country-limiter/ for full documentation.

== Installation ==

1. Upload the entire 'payment-gateways-country-limiter' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
